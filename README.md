# Hop is a simple cli tool for jumping between favorite directories.

## Commands

- `hop <shortcut>`: hop to a directory.
- `hop add <shortcut>` | `hop a <shortcut>`: Add a current directory.
- `hop list` | `hop ls`: List all your saved directories.
- `hop remove <shortcut>` | `hop rm <shortcut>`: Remove a shortcut entry.
- `hop fuzzy` | `hop f`: hop to a directory using fuzzy selection from list.
